# 📱 Google Play Store Data Analysis

## 📌 Project Overview
This project focuses on analyzing the **Google Play Store dataset** using Python to uncover meaningful insights about app categories, ratings, reviews, installs, pricing, and user trends.

The goal of this project is to apply **data cleaning, exploratory data analysis (EDA), visualization, and hypothesis testing** to understand app performance and market patterns in the Play Store.

---

## 🎯 Objectives
- Clean and preprocess the dataset
- Explore app-related features and trends
- Perform Exploratory Data Analysis (EDA)
- Visualize app statistics using charts and graphs
- Identify patterns in ratings, installs, and categories
- Perform statistical/hypothesis testing for insights

---

## 📂 Project Workflow

### 1. Data Cleaning
- Removed duplicates
- Handled missing values
- Cleaned columns like:
  - Installs
  - Price
  - Reviews
  - Size
- Converted data into proper formats for analysis

### 2. Data Exploration
Explored important features such as:
- App Category
- Rating
- Reviews
- Size
- Installs
- Type (Free/Paid)
- Content Rating
- Genres

### 3. Exploratory Data Analysis (EDA)
Performed analysis to answer questions such as:
- Which app categories are most common?
- Which categories have the highest ratings?
- Do free apps dominate the market?
- Which apps receive the highest installs?
- Is there a relationship between rating and number of installs?

### 4. Data Visualization
Created charts and visual representations such as:
- Bar Charts
- Count Plots
- Histograms
- Pie Charts
- Scatter Plots
- Box Plots
- Heatmaps

### 5. Hypothesis Testing
Applied statistical methods to test assumptions and discover meaningful relationships in the dataset.

---

## 📈 Key Insights
Some major insights from this project include:
- Certain app categories dominate the Play Store
- Free apps are significantly more common than paid apps
- Some categories have better average ratings than others
- Apps with more installs often have higher visibility and engagement
- Ratings and reviews provide useful indicators of app performance
